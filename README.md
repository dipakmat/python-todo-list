# To-Do List App

A simple Python-based To-Do List application.

## Usage

1. Install Python.
2. Clone the repository.
   ```
   bash git clone <repository_url>
   cd todo-list
   ```
3. Run the application.
   ```
   bash python todolist.py
   ```
